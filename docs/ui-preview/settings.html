Settings

独立设置页用于管理本项目个人配置。保存后立即生效，并返回已重载模块和仅下一次运行生效的字段。

独立页面 当前路由：/docs/ui-preview/settings.html

配置说明

  • Storage：工具运行数据写入路径，更新后仅影响后续新写入，不自动迁移历史数据。
  • Workspace：被测/被改项目目录与 Git 根识别策略，影响 run 与 code task 执行上下文。
  • Test Assets：共享测试目录和候选测试目录，控制默认是否纳入运行。
  • Diagnostics：关联键提取规则和时间窗口，影响 trace/log 查询命中率。
  • Trace / Logs：观测后端 provider 与 endpoint，保存后查询类能力立即生效。
  • AI / CodeAgent：模型与修复策略，影响新任务默认行为；部分字段可能仅下一次 run 生效。
  • Report / UI：报告端口和语言；端口变更仅在下次启动生效。
